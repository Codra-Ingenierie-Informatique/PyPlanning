How to install
--------------

Wheel package:
^^^^^^^^^^^^^^

On any operating system, using pip and the Wheel package is the easiest way to
install PyPlanning on an existing Python distribution:

.. code-block:: console

    $ pip install --upgrade PyPlanning-1.0.0-py2.py3-none-any.whl


Source package:
^^^^^^^^^^^^^^^

Installing PyPlanning directly from the source package is straigthforward:

.. code-block:: console

    $ python -m build
